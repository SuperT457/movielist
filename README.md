# movielist
A simple web application to track movies you want to watch.
It's based on:
1. Postgreql
2. expressjs (for front-end)
3. node.js and html/css (for a simple frontend)

As you see from setup, it's a really simple basic application that I used to practice.
It includes authentication, an, after that, a simple list of movies that you can edit using REST APIs.
